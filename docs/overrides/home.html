Distributed data.
Local control.

Neurobagel aims to connect decentralized neuroimaging datasets using linked data principles, while allowing data files and access control to remain locally with data owners.

Get started Learn more →

Federated data integration

Our aim is to let researchers and data owners harmonize and integrate their data without the need to upload them to a central database. We are building an ecosystem of tools for local data harmonization and management, that allows users to search for cohorts of participants across datasets.

Your data, your rules
You always retain all of your data. This avoids the usual vendor lock-in of centralized databases.
Open standards
Data are annotated using existing and widely adopted controlled vocabularies and are represented using open web standards. This allows data to be easily integrated with other resources.
